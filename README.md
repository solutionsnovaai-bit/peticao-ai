# PETIÇÃO.AI — Inteligência Jurídica de Elite

Landing page de alta conversão para automação de petições jurídicas com IA.

## Stack

- React 19 + TypeScript
- Vite 6
- Tailwind CSS v4
- GSAP + SplitType (animações)
- Canvas confetti

## Rodar localmente

```bash
npm install
npm run dev
```

Acesse em: http://localhost:3000

## Deploy no Vercel

1. Faça push deste repositório no GitHub
2. Importe no [vercel.com](https://vercel.com)
3. **Framework preset:** Vite
4. **Build command:** `npm run build`
5. **Output directory:** `dist`
6. Clique em Deploy — pronto.

> Não é necessária nenhuma variável de ambiente para o deploy desta landing page.
